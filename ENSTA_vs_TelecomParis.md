# Comparaison IA — Télécom Paris vs ENSTA Paris

## Verdict synthétique

| Question | Réponse |
|---|---|
| **Classement IA / informatique global** | **Télécom Paris légèrement devant ENSTA Paris** |
| **Classement prestige général école** | **ENSTA Paris devant Télécom Paris** selon L’Étudiant 2026 |
| **Écart réel en IA** | Faible : les deux sont dans le même écosystème IP Paris / Palaiseau et partagent une partie de l’offre IA |
| **Pourquoi Télécom Paris passe devant en IA** | École plus naturellement centrée numérique, informatique, data, réseaux, IA, software, GenAI, cloud, cybersécurité |
| **Pourquoi ENSTA reste très compétitive** | Meilleur rang général, très forte sélectivité, parcours IA avancée, robotique, agents autonomes, vision, systèmes complexes |
| **Conclusion courte** | **Télécom Paris** si elle vise IA moderne / software AI / GenAI / data / Big Tech ; **ENSTA Paris** si elle vise IA scientifique / robotique / systèmes autonomes / deeptech / défense |

## Conclusion courte

Je mettrais **Télécom Paris légèrement devant ENSTA Paris pour l’IA**, surtout si l’objectif est **IA moderne, GenAI, NLP, software AI, data science, ML engineering, cloud ou Big Tech**.

Je mettrais **ENSTA Paris devant Télécom Paris** pour **prestige général**, **systèmes autonomes**, **robotique**, **vision**, **défense/deeptech**, et pour un profil IA plus scientifique / systèmes complexes.

Donc :

1. **Télécom Paris** — meilleur choix IA / informatique / GenAI / software / data.
2. **ENSTA Paris** — meilleur choix prestige général + IA systèmes / robotique / deeptech.
