# Comparaison IA — Centrale Nantes vs Télécom SudParis

## Verdict synthétique

| Question | Réponse |
|---|---|
| **Classement IA global** | **Centrale Nantes légèrement devant Télécom SudParis** |
| **Écart réel** | Faible à modéré |
| **Pourquoi Centrale Nantes reste devant** | Meilleur prestige généraliste, rang L’Étudiant nettement supérieur, option IA très claire, meilleure image généraliste post-prépa. |
| **Pourquoi Télécom SudParis est très compétitive** | VAP IA commune avec Télécom Paris / ENSTA à Palaiseau, MAIA très appliquée / MLOps / production, doubles diplômes Imperial + Georgia Tech. |
| **Conclusion courte** | Centrale Nantes > Télécom SudParis en classement global ; Télécom SudParis peut passer devant si l’objectif est IA appliquée, MLOps, data engineering, cloud, ou double diplôme Georgia Tech / Imperial. |

## Conclusion courte

**Je mettrais Centrale Nantes légèrement devant Télécom SudParis en classement IA global**, parce que le prestige général, le rang L’Étudiant, et l’option InfoIA très solide donnent un meilleur signal d’ensemble.

Mais **Télécom SudParis peut être le meilleur choix** si elle vise une IA plus opérationnelle : **ML engineering, MLOps, cloud, data engineering, NLP appliqué, IA générative en production**, ou si elle accorde beaucoup d’importance aux doubles diplômes **Georgia Tech / Imperial**.

Donc mon classement serait :

1. **Centrale Nantes** — meilleur compromis généraliste + IA + prestige.
2. **Télécom SudParis** — meilleur pour IA appliquée / production / MLOps / international US-UK.
